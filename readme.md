
# Simple bar chart

A simple HTML demo on how to create a barchart without the need for a 
third party library.

## Set up

The environment.

```sh
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Create a `.flaskenv` file if you like automatic reloads.

```sh
# file: .flaskenv
FLASK_ENV=development
FLASK_RUN_PORT=5000
``` 

## Run

```sh
source venv/bin/activate
flask run
```
